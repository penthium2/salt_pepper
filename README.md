# salt_pepper
Brute force encrypted file with cipher and pass

### Usage 
```
Usage: salt-pepper [-f SOURCE FILE] [-p PASS ]

Brute force encrypted file with cipher and pass

	-f	encrypted source file 
	-p	passphrase 
	-h	display help
	-v	display version
```
All files are saved in the **FLAGS** directory.

### Example
```
./salt-pepper -f salted-file -p "PASSWORD"
```
